Pew game A simple little two-player game made in JavaScript with no external libraries. Controls: Player 1 - Arrow keys + mouse; Player 2 - WASD + Space. Made in a couple hours at MasseyHacks V for our group webpage.

CSC207 Final Project (GitHub) A group project resulting in a calendar application. The project was written in Java and I used Git for the first time. Some functionality is broken or not fully finished. Users can create new accounts, login, create multiple calendars, add events to calendars, search calendars by certain attributes, etc. I was personally responsible for the GUI programming, implementing many of the layouts found in JavaX and using external libraries to assist with, for example, selecting a date.

Webpage Source (GitHub) All the code and resources used in the website you are currently viewing.

MasseyHacks V Project (GitHub) A group project made for MasseyHacks V in 48 hours. The objective was to create something to keep our world's water clean. Our approach was to make a website to spread awareness about the importance of water. This was accomplished through creating games to target kids and a calculator to show how much water consumption an individual actually has.

Grade 12 Statistics Final Project The written portion of my Grade 12 Statistics course. I really think this is worth including since the content is not only very interesting to me, but demonstrates proficiency in manipulating spreadsheet data, data visualization/presentation, and LaTeX. The goal was to identify a method for correlating various factors (net migration, education spending, population, homicide rate) with GDP per capita.
